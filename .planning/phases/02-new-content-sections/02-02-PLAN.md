**Static data — tech categories:**
```ts
import {
  SiShopify,
  SiNextdotjs,
  SiReact,
  SiTypescript,
  SiTailwindcss,
  SiFramer,
  SiStripe,
  SiGoogleanalytics,
} from "react-icons/si";
import type { IconType } from "react-icons";

interface Tech {
  name: string;
  Icon: IconType;
}

interface Category {
  label: string;
  items: Tech[];
}

const categories: Category[] = [
  {
    label: "Frontend",
    items: [
      { name: "Next.js", Icon: SiNextdotjs },
      { name: "React", Icon: SiReact },
      { name: "TypeScript", Icon: SiTypescript },
      { name: "Tailwind CSS", Icon: SiTailwindcss },
      { name: "Framer Motion", Icon: SiFramer },
    ],
  },
  {
    label: "Ecommerce",
    items: [
      { name: "Shopify", Icon: SiShopify },
      { name: "Stripe", Icon: SiStripe },
    ],
  },
  {
    label: "Analytics",
    items: [
      { name: "Google Analytics", Icon: SiGoogleanalytics },
    ],
  },
];
```

**Animation variants:**
```ts
const containerVariants = {
  hidden: { opacity: 0 },
  visible: {
    opacity: 1,
    transition: { staggerChildren: 0.08, delayChildren: 0.1 },
  },
};

const itemVariants = {
  hidden: { opacity: 0, y: 20 },
  visible: { opacity: 1, y: 0, transition: { duration: 0.4 } },
};
```

**Layout:**
- Section: `py-32 border-y border-white/5 bg-white/[0.02]` (alternating section tint — matches Stats section pattern)
- Heading block: `motion.div initial={{ opacity: 0, y: 30 }} whileInView={{ opacity: 1, y: 0 }} transition={{ duration: 0.6 }} viewport={{ once: true }}`
- Heading: `text-4xl md:text-5xl font-bold mb-6 text-neon-white` — "Our Stack"
- Accent bar: `h-1 w-20 bg-primary mb-6`
- Subtitle: `text-xl text-gray-400` — "Precise tools, chosen for performance and longevity."

**Category rendering:**
```tsx
<div className="space-y-12">
  {categories.map((cat) => (
    <div key={cat.label}>
      <p className="text-xs font-semibold uppercase tracking-widest text-gray-500 mb-4">
        {cat.label}
      </p>
      <motion.div
        variants={containerVariants}
        initial="hidden"
        whileInView="visible"
        viewport={{ once: true }}
        className="flex flex-wrap gap-3"
      >
        {cat.items.map(({ name, Icon }) => (
          <motion.div
            key={name}
            variants={itemVariants}
            className="flex items-center gap-2 px-4 py-2 rounded-full
                       bg-white/5 border border-white/10
                       hover:border-primary/40 hover:neon-glow-cyan
                       transition-all duration-300 cursor-default"
          >
            <Icon className="text-primary text-lg flex-shrink-0" />
            <span className="text-sm font-medium text-gray-300">{name}</span>
          </motion.div>
        ))}
      </motion.div>
    </div>
  ))}
</div>
```

**Critical rules:**
- All react-icons imports must be NAMED imports: `import { SiShopify } from "react-icons/si"` — NOT default imports (v5 breaking change)
- Icon color: use `className="text-primary"` (sets CSS `color` → SVG `currentColor`) — do NOT use `fill-primary` (may be undefined in Tailwind v4)
- `SiFramer` is correct — `SiFramermotion` does NOT exist in react-icons
- Do NOT add `whileHover` to badge items — use CSS `hover:neon-glow-cyan` only (prevents staggerChildren conflict)
- Each category has its own `motion.div` stagger container so each category's badges animate independently
- `viewport={{ once: true }}` on every stagger container
