**Animation variants (containerVariants + itemVariants pattern from research):**
```ts
const containerVariants = {
  hidden: { opacity: 0 },
  visible: {
    opacity: 1,
    transition: { staggerChildren: 0.15, delayChildren: 0.1 },
  },
};

const itemVariants = {
  hidden: { opacity: 0, y: 24 },
  visible: { opacity: 1, y: 0, transition: { duration: 0.5 } },
};
```

**Section layout:**
- Section element with `py-32 bg-background` (matching existing Services section pattern)
- Heading block: `motion.div` with `initial={{ opacity: 0, y: 30 }} whileInView={{ opacity: 1, y: 0 }} transition={{ duration: 0.6 }} viewport={{ once: true }}` — matches `app/page.tsx` heading pattern exactly
- Heading: `text-4xl md:text-5xl font-bold mb-6 text-neon-white` with text "How We Work"

**Timeline row:**
```tsx
<motion.div
  variants={containerVariants}
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true }}
  className="flex flex-col md:flex-row items-start gap-0"
>
  {steps.map((step, i) => (
    <React.Fragment key={step.number}>
      <motion.div
        variants={itemVariants}
        className="flex-1 min-w-0 p-6 md:p-8 rounded-3xl bg-white/[0.03] border border-white/10
                   hover:border-primary/40 hover:neon-glow-cyan transition-all duration-300"
      >
        <span className="block text-5xl md:text-6xl font-bold text-primary/30 mb-3 leading-none">
          {step.number}
        </span>
        <h3 className="text-xl font-bold text-white mb-3">{step.title}</h3>
        <p className="text-gray-400 text-sm leading-relaxed">{step.description}</p>
      </motion.div>
      {i < steps.length - 1 && (
        <div className="hidden md:flex items-center flex-shrink-0 text-primary/30 text-xl px-3 self-center">
          ──▶
        </div>
      )}
    </React.Fragment>
  ))}
</motion.div>
```

**Critical rules (from research pitfalls):**
- Do NOT add `whileHover` to the step cards — use CSS hover classes only (`hover:border-primary/40 hover:neon-glow-cyan`)
- Connector `div` must have `flex-shrink-0` to prevent squeezing at md breakpoints
- Step cards must have `flex-1 min-w-0` so they share available space equally
- Parent must have both `initial="hidden"` AND `whileInView="visible"` — do not omit initial
- `viewport={{ once: true }}` on the parent container — consistent with project pattern

**Imports needed:** `"use client"`, `motion` from `"framer-motion"`, `React` from `"react"` (for React.Fragment)
